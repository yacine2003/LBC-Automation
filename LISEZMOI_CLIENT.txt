========================================
  LBC AUTOMATION - GUIDE CLIENT
========================================

PRÉREQUIS :
- Windows 10/11
- Connexion Internet

========================================
INSTALLATION (À FAIRE UNE SEULE FOIS)
========================================

1. Télécharger Python :
   - Allez sur : https://www.python.org/downloads/
   - Téléchargez la dernière version (3.10 ou supérieur)
   - Lors de l'installation : COCHEZ "Add Python to PATH"
   - Installez Python

2. Installer LBC Automation :
   - Double-cliquez sur : INSTALLER_CLIENT.bat
   - Attendez la fin de l'installation (2-5 minutes)

3. Configurer :
   - Vérifiez que le fichier service_account.json est présent
   - Double-cliquez sur : LANCER.bat
   - Une page web s'ouvre automatiquement
   - Cliquez sur "Configuration"
   - Remplissez vos informations :
     * Identifiants Leboncoin
     * Nom du Google Sheet
     * Dossier des photos
   - Enregistrez

========================================
UTILISATION QUOTIDIENNE
========================================

1. Double-cliquez sur : LANCER.bat
2. Une page web s'ouvre
3. Cliquez sur "Démarrer"
4. Suivez la progression en temps réel

Pour arrêter : Cliquez sur "Arrêter" ou fermez la fenêtre

========================================
STRUCTURE DU GOOGLE SHEET
========================================

Votre Google Sheet doit contenir ces colonnes :

| Titre | Description | Prix | Catégorie | Photos | Statut | Type | Ville |

- Statut : Mettez "A_FAIRE" pour publier une annonce
- Photos : Noms des fichiers séparés par des virgules (ex: "photo1.jpg,photo2.jpg")

N'oubliez pas de partager votre Google Sheet avec :
lbc-automation@lbc-automation-483321.iam.gserviceaccount.com

========================================
FICHIERS IMPORTANTS
========================================

- INSTALLER_CLIENT.bat : Installation (une seule fois)
- LANCER.bat : Lancer le bot (quotidien)
- service_account.json : Authentification Google Sheets (ne pas supprimer)
- config.env : Configuration sauvegardée (créé automatiquement)

========================================
EN CAS DE PROBLÈME
========================================

Problème : "Python n'est pas reconnu"
Solution : Réinstallez Python en cochant "Add Python to PATH"

Problème : "service_account.json manquant"
Solution : Vérifiez que ce fichier est dans le même dossier

Problème : "Impossible de se connecter au Google Sheet"
Solution : Vérifiez que le Sheet est partagé avec l'email du service account
